#### How can I write comments in JSX? <a href="#how-can-i-write-comments-in-jsx" id="how-can-i-write-comments-in-jsx"></a>

```jsx
<div>
  {/* Comment goes here */}
  Hello, {name}!
</div>
```

```jsx
<div>
  {/* It also works 
  for multi-line comments. */}
  Hello, {name}!
</div>
```
